    ASTTool

    Passed to Transformations to aid in transversing and modifying the AST

    This should really only be generated by a Transformer unless you know what you're doing.

    This offers a range of functions to help in modifying an AST fragment. This is offered to all transformations by a Transformer object and will provide primarially the modification ability along with other common tasks. This serves as an interface between the AST and a transformation.

    This provides the necessary abstraction needed in order to prevent accidental mutations and verify that the AST is indeed being properly transversed.
